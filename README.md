## 💻 INSTALLATION

### 🦊 FIREFOX
Available directly on the Firefox Add-ons store. No manual load required.
👉 **[GET FOR FIREFOX](https://addons.mozilla.org/addon/leettracker/)**

### 🌐 CHROME / CHROMIUM
1. **DOWNLOAD** the latest `.zip` release from [GitHub Releases](https://github.com/PrasanthPradeep/LeetTracker/releases)
2. **EXTRACT** the downloaded archive.
3. **OPEN** `chrome://extensions` in your browser.
4. **ENABLE** `Developer mode` (toggle at top-right).
5. **CLICK** `Load unpacked` and select the extracted folder.
